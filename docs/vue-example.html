Vue

Spin component bardzo sprawnie integruje się w aplikacjach zbudowanych we Vue. Używanie go nie wiąże się z żadnymi specjalnymi działaniami ze strony developera.
QR kode
Implementacja w HTML. We frame worku VUE developer składa aplikację z komponentów, które sobie definiuje. Component TAU Spin jest definiowany jak każdy inny komponent we VUE i nie wymaga od developera żadnej specyficznej wiedzy. Konfiguracja komponentu na potrzeby developera odbywa się przez ustawienie żądanych parametrów.
