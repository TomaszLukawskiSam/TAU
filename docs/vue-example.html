Vue

TAU component integrates very efficiently in applications built in Vue. Using it does not involve any special actions on the part of the developer. Example
Implementation. In the VUE framework, the developer assembles the application from components that he defines. For example, component TAU Spin is defined like any other component in VUE and does not require from the developer no specific knowledge. Component configuration for needs developer is done by setting the desired parameters.
