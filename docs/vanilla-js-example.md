## TAU Components > Vanilla JS example

### Introduction

A basic example of use based only on the browser API.
The Spin component as an independent component.

### Code

```html
<tau-spin class="default-spin"
	min="1900"
	max="2200"
	value="2000"
	loop="enabled"
	roll-height="custom"
	item-height="50"
	momentum-level="1"
	duration="500">
</tau-spin>
```
